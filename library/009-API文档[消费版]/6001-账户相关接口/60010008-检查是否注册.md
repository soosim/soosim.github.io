## 接口简介
检查是否注册

## 接口详情

### 请求地址
测试：http://shopping.api.test.soolife.cn/v2/account/check/{loginId}

正式：http://shoppingapi.soolife.cn/v2/account/check/{loginId}

### 请求类型
post

### 请求参数 (Form Data)
| 参数名     |   类型   |  必填  | 描述           | 默认值  | 参考值  |
| ------- | :----: | :--: | ------------ | ---- | ---- |
| loginId | string |  是   | 手机号/用户名/邮箱地址 | -    | -    |

### 返回正确JSON示例
```javascript
状态码：200
{
  "success": true,
  "msg": "用户已存在！"
}
```
### 返回错误JSON示例
```javascript
状态码：400
{
  "success": false,
  "msg": "用户未注册！"
}
```

### 其它状态码错误
| 状态码  | 错误原因 |
| :--- | :--- |
| 无    | -    |
