## 接口简介
忘记密码

## 接口详情

### 请求地址
测试：http://shopping.api.test.soolife.cn/v2/account/forgot/vcode

正式：http://shoppingapi.soolife.cn/v2/account/forgot/vcode

### 请求类型
post

### 请求参数 (Form Data)
| 参数名   |   类型   |  必填  | 描述    | 默认值  | 参考值         |
| ----- | :----: | :--: | ----- | ---- | ----------- |
| phone | string |  是   | 用户手机号 | -    | 18212345678 |
| vcode | string |  是   | 验证码   | -    | 263814      |

### 返回正确JSON示例
```javascript
状态码：200
{
  "code": 106,
  "msg": "验证通过!"
}
```
### 其它状态码错误
| 状态码  | 错误原因     |
| :--- | :------- |
| 100  | 手机号码不正确  |
| 101  | 短信验证码错误! |
| 106  | 验证通过!    |
