# 根据位置 列出活跃的广告
>维护人员：**孙单单**
>创建时间：2017-3-31

## 接口简介
根据位置 列出活跃的广告

## 接口详情

### 请求地址
http://v1.api.soolife.cn/ads/lively/{code}

### 请求类型
GET

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
| code | string | 是 | 广告位置编号 | - | 如：app.supermarket.meeting |
