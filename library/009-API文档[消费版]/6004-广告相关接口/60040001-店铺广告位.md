# 广告位置接口文档
>维护人员：**Xie Jinlong**  
>创建时间：2017-3-17

## 接口简介
针对单个店铺的广告信息  

## 接口详情

### 请求地址
http://v1.api.soolife.cn/ads/shop/location/{id}

### 请求类型
GET

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
| id | int | 是 | 店铺id | - | 220 |
