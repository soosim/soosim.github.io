## 接口简介
APP满额抽奖活动商品（列表）

## 接口详情

### 请求地址
http://api.test.soolife.net/fulldraw/fullgoods/{fullid:[0-9]+}/{fcid:[0-9]+}

### 请求类型
POST

### 请求参数 (Form Data)
| 参数名    |  类型  |  必填  | 描述   | 默认值  | 参考值  |
| ------ | :--: | :--: | ---- | ---- | ---- |
| fullid | int  |  必填  | 活动编号 | -    | 1    |
| fcid   | int  |  必填  | 类型编号 | -    | 1    |
