## 接口简介
星币活动列表页

## 接口详情

### 请求地址
http://shoppingapi.test.soolife.net/coin/activity

### 请求类型
POST

### 请求参数 (RAW)
```javascript
{
  "skip":"0", 		// 分页跳过条数
	"take":"10", 		// 分页显示条数
	"category":"1", 	// 分类 1-衣 2-食 3-住 4-行 5-娱 else 全部
	"just_coin":true, 	// true-星兑换 false-星换购
	"coin_min":"10", 	// 最小星币数
	"coin_max":"100" 	// 最大星币数
}
```
