## 接口简介
二级频道页 列表

## 接口详情

### 请求地址
http://shoppingapi.test.soolife.cn/v2/channel/list/{category:[1-5]}
http://shoppingapi.soolife.cn/v2/channel/list/{category:[1-5]}

### 请求类型
POST

### 请求参数 (Form Data)
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
|category|int|是|分类|-|1衣  2食  3住   4行   5娱|
###body体中的数据
```javascript
{
    "type":"1",//1 已开始的限时抢购 2 下期预告的显示抢购
    "size":"10",//一页中含有多少条数据
    "index":"1"//当前页码
}
```
