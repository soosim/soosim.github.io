## 接口简介
海外精品的列表页

## 接口详情

### 请求地址
    测试正式的环境：http://shoppingapi.test.soolife.cn/v2/overseas/goods/list
    正式的环境：http://shoppingapi.soolife.cn/v2/overseas/goods/list

### 请求类型
POST

### 请求参数 (Form Data)
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
|无|-|-|-|-|-|
###body体中的数据
```javascript
{
    "size":"10",//一页中含有多少条数据
    "index":"1"//当前页码
}
```
