## 接口简介
订单售后服务完成

## 接口详情

### 请求地址
http://v1.api.soolife.cn/orders/service/solved/{id:[0-9]+}

### 请求类型
get

### 请求参数 (Form Data)
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
|id|int|是|售后处理ID|-|如：1142|

### 返回正确JSON示例
```javascript
{
  "success": true,
  "msg": "提交信息成功!"
}
```
### 返回错误JSON示例
```javascript
{
  "success": false,
  "msg": "当前订单正在处理，不要着急哦！"
}
```

### 其它状态码错误
| 状态码 | 错误原因     |
| :------------- | :------------- |
|405|请求的方法不支持|
