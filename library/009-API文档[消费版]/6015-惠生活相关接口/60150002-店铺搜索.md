## 接口简介

## 接口详情

### 请求地址
http://life.api.soolife.cn/v1/store/search

### 请求类型
POOT

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
{

	"latitude":" 32.242695",
	"longitude":"122.962011",
	"orderby":[distance],
	"category":"[ F A B S ]",
	"area": 区域编号,
	"type":"[1 体验店] [2 外部商家]",
	"keyword":"关键字",
	"skip":"2",
	"take":"5"
}
