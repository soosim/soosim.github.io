### 请求地址
http://api.test.soolife.net/starpink/newstarpink/{type}/{size}

### 请求类型

| 参数名  |  类型  |  必填  |   描述   | 默认值  | 参考值  |
| :--: | :--: | :--: | :----: | :--: | :--: |
| type | int  | true | 1最新2最热 |  1   |      |
| size | int  | true |   长度   |  10  |      |
