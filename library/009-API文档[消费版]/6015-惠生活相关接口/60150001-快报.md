## 接口详情

### 请求地址
http://life.api.soolife.loc/v1/expressnews

### 请求类型
GET

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
  id

### 返回正确JSON示例
```JavaScript
{
  "state": {
    "code": "200",
    "msg": "ok"
  },
  "data": [
    {
      "id": "4",
      "title": "q",
      "b_begindate": "1490544000",
      "i_time": "1491465443",
      "creater": "tony"
    },
    {
      "id": "5",
      "title": "wwww",
      "b_begindate": "1490630400",
      "i_time": "1491465446",
      "creater": "tony"

}
```
### 返回错误JSON示例
```JavaScript
{
    "state": {
        "code": 10500
        "msg": "服务器未知报错"
    }
}
```

### 其它状态码错误

| 400  | 请求失败    |
