## 接口简介
删除收货地址

## 接口详情

### 请求地址
http://v1.api.soolife.cn/address/delete/{addressid:[0-9]+}

### 请求类型
delete

### 请求参数 (Form Data)
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
|addressid|int|是|地址ID|-|如：2356|

### 返回正确JSON示例
```javascript
{
  "success": true,
  "msg": "删除成功!"
}
```
### 返回错误JSON示例
```javascript
{
    "success": false,
    "msg": "请求方法不支持"
}
```
