# 获取店铺热销商品
>创建人员：**孙单单**
>创建时间：2017-03-31

[TOC]

## 接口简介
获取店铺热销商品

## 接口详情

### 请求地址
http://api.test.soolife.net/member/favorite/shop/{shop_id}
### 请求类型
delete  请求表示取消关注   post  请求表示关注

### 请求参数 (Form Data)
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
|无|-|-|-|-|-|

### 返回正确JSON示例
```javascript
delete：
{
  "success": true,
  "msg": "取消关注成功"
}
post：
{
  "success": true,
  "msg": "关注成功"
}
```
### 返回错误JSON示例
```javascript
delete：
{
  "success": false,
  "msg": "您已经取消过关注过本店铺了"
}
post：
{
  "success": false,
  "msg": "您已经关注过本店铺了"
}
```

### 其它状态码错误
| 状态码 | 错误原因     |
| :------------- | :------------- |
|403|当前用户已退出登录,请重新登录|
|400|参数不合法|
