## 接口简介
活动列表，xxx

## 接口详情

### 请求地址
http://api.test.soolife.net//starkill/lists/{type}/{skip}/{take}

### 请求类型
GET

### 请求参数
| 参数名  | 类型   | 必填   | 描述                                  | 默认值  | 参考值     |
| ---- | ---- | ---- | ----------------------------------- | ---- | ------- |
| type | int  | true | 1正在进行 2 即将开始 3我的列表(继续杀) 4 我的列表 (购买) | 1    | 1/2/3/4 |
| skip | int  | true | 起步                                  |      |         |
| take | int  | true | 步长                                  |      |         |

  skip | take
