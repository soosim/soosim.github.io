## 接口简介
获取最近注册奖励内容

## 接口详情

### 请求地址
http://v1.api.soolife.cn/member/registeraward/{no:\w+}/{type:[0-9]+}

### 请求类型
get（不需要token）

### 请求参数 (Form Data)
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
|no|string|是|注册来源|-|如：wallet201611|
|type|int|是|注册还是分享|1|-|

### 返回正确JSON示例
```javascript
{
  "is_referrer": "0",
  "coin": "0",
  "money": "20",
  "cash": "0",
  "couponNo": "0",
  "content": "领取红包20元"
}
```
