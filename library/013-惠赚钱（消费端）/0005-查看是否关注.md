## 接口简介
查看是否关注星店铺

## 接口详情

### 请求地址
http://money.api.test.soolife.cn/store/isfocus/{$store_id}

### 请求类型
get

### 请求参数 (Form Data)
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
|-|-|-|-|-|-|

### 返回正确JSON示例
```javascript
{
  "store_id": "1621",
  "member_id": "3000876",
  "is_focus": 1 *已关注*
}
```
### 返回错误JSON示例
```javascript
{
  "store_id": "1621",
  "member_id": "3000876",
  "is_focus": 0 *未关注*
}
{
  "success": false,
  "msg": "当前用户已退出登录,请重新登录"
}
```
