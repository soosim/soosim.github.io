## 接口简介
商品管理页 商品列表

## 接口详情

### 请求地址
http://money.api.test.soolife.net/good/manage/list

### 请求类型
post

### 请求参数 (raw)
```javascript
{
	"index":"", 当前页
	"size" :"", 每页条数
	"category":"", 分类 衣食住行娱 12345
	"is_putaway":"1", 出售中
	"status":"1" 批量管理用
}
```
