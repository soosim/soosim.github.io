## 接口简介
商品添加页 商品列表

## 接口详情

### 请求地址
http://money.api.test.soolife.net/good/rebatelists

### 请求类型
post

### 请求参数 (raw)
```javascript
{
	"index":1, 第几页
	"size" :10, 每页多少
	"category":"", 分类 空为全部 衣食住行娱 12345
	"profit":"" 利润 1 = 1-10%,2 = 10-20%,3 = 20-30%, 4 = 30%以上
}
```
