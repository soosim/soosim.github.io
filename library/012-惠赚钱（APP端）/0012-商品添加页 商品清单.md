## 接口简介
商品添加页 商品清单

## 接口详情

### 请求地址
http://money.api.test.soolife.net/good/goodsrepertoire

### 请求类型
post

### 请求参数 (raw)
```javascript
{
	"skip" : "0", //默认跳过0条
	"take" : "20" //默认8条
}
```
