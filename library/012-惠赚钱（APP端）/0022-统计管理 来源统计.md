## 接口详情

### 请求地址
http://money.api.test.soolife.net/source/stat/{time}

### 请求类型
get

### 请求参数
```javascript
"time" : //1.七天，2.30天,3.60天,4.半年,5.一年
```

```javascript
{

    "2017-04-23",
    "2017-04-22",

    "2017-04-20",
    "2017-04-19",

    "2017-04-17"

    "rests": 0,
    "wechat": 12,

  }
}
```
